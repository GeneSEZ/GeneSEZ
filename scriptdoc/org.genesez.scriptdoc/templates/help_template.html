How This API Document Is Organized

This API (Application Programming Interface) document has pages corresponding to the items in the navigation bar, described as follows.

Overview

The Overview page provides a list of all scanned scripts with the specified headcomment for each. Furthermore, it can be used to get a detailled overview of an oAW-script. To be more exactly, it shows in the begining the namespace, the Eclipse project-path and the directory of the current script. Moreover, the headcomment, headimport and headextensions are shown. Afterwards, there is a summary of all executables contains in this script, redescribed below.

NameSpace

The Namespace document is the front page of this API document and provides a list of all scanned namespaces.

MetaType

Every object has a type. The Metatype page shows all scanned types, furthermore it shows all executables of a specified type.

Index

The Index page contains an alphabetic list of all executables.

Stats

The Stats consists out of a list of calculated code metrics. To be more accurate, it shows some line-metrics, the cyclomatic complexity of McCabe and some chosen Halstead-metrics.
